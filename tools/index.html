Online tools

Some (hopefully useful) online tools I've developed over the years. Everything is open source here.

  • Visual Protobuf

    Tool to visually inspect bytes in a serialied protobuf message.

  • Markdown table-of-contents generator

    Paste you Markdown, instantly get a table of contents.

  • Tilix layout builder

    Generates SSH downloadable layouts for the Tilix terminal.

  • JWT base 64 decoder

    Helps decoding JWT tokens.

  • Gear ratio calculator

    Calculate your bicycle's gear ratios online.

  • Column editor

    Simple tool for pasting stuff in column mode.

  • Calculadora de salário CLT (in portuguese)

    Antiga calculadora que fiz para ajudar a entender as diferenças entre CLT e PJ (valores desatualizados para os dias atuais).
